# Дискомфорт — точка роста

Одностраничный сайт программы трансформации. Готов к публикации через GitHub Pages.

## Структура

```
index.html            — сам сайт
assets/hero-editorial.jpeg  — фото в шапке
README.md              — этот файл
```

## Как выложить на GitHub Pages

1. Создай новый репозиторий на [github.com/new](https://github.com/new) (например, `moy-sayt`).
2. Загрузи в него все файлы из этой папки, сохранив структуру (`index.html` в корне, `assets/` — папка рядом с ним).
   - Проще всего: на странице репозитория нажми **Add file → Upload files** и перетащи всё сразу.
3. Перейди в **Settings → Pages**.
4. В разделе **Build and deployment** выбери источник **Deploy from a branch**, ветку `main`, папку `/ (root)` → **Save**.
5. Через 1–2 минуты сайт будет доступен по адресу:
   `https://<твой-логин>.github.io/<название-репозитория>/`

## Важно

- Файл `index.html` должен называться именно так — GitHub Pages ищет его по умолчанию.
- Не переименовывай папку `assets` и не перемещай фото — иначе оно пропадёт с сайта (путь к нему прописан в коде как `assets/hero-editorial.jpeg`).
- Кнопка "Написать в Telegram" ведёт на `t.me/Alexsiop` — если нужно поменять, ищи это в `index.html`.
